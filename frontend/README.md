# 🧠 AI Exam Proctoring System

A real-time AI-based exam monitoring system using MERN + Socket.IO + Edge AI.

---

## ⚡ Features

- Face Detection (face-api.js)
- Gaze Tracking
- Audio Noise Detection
- Behavioral Biometrics
- Real-time Risk Scoring AI
- DevTools / Copy-Paste Protection
- Multi-tab detection
- Live Admin Dashboard
- Forensic Timeline + Reports

---

## 🧠 Architecture

Frontend → React + Vite + Tailwind  
Backend → Node.js + Express  
Database → MongoDB  
Realtime → Socket.IO  
Cache/Queue → Redis
